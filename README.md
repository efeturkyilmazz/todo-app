# To-Do List Uygulaması

Bu proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit bir yapılacaklar listesi (to-do list) uygulamasıdır.

##  Özellikler

-  Yeni görev ekleme  
-  Görev silme  
-  Görev tamamlama
